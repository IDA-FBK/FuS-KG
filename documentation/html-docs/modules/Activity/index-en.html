Activity Module Specification

Abstract

The activity module imports the Core and the Multimodal Modules and it includes the taxonomy of all activities covered by FuS-KG together with the knowledge related to the effort required to fulfill each activity. The effort is represented through the Metabolic Equivalent of Task (MET) coefficient and the calories required to perform 1 minute of each activity for each kilogram of body weight.

Introduction back to ToC

The activity module imports the Core and the Multimodal modules and it includes the taxonomy of all activities covered by FuS-KG together with the knowledge related to the effort required to fulfill each activity. The Activity category is divided into two categories (i.e., LifeActivity and PhysicalActivity) that are further sub-categorized. Then, each activity is connected to a SpecializedActivity through the specializes property. The SpecializedActivity concept identifies activities specifying their metabolic effort based on who performs the activity. In particular, a SpecializedActivity is connected to a PractitionerType with the practicedBy property which indicates the type of person that performs the activity (i.e., an adult, an older adult, or a wheelchair user), and the effort is represented twofold through the Metabolic Equivalent of Task (MET) coefficient using the hasMETValue data property and the calories required to perform 1 minute of each activity for each kilogram of body weight using the hasUnitKCalValue data property.

Activity Module of Functional Status Knowledge Graph (FuS-KG): Description back to ToC


The Activity module imports the Core and the Multimodal module. It introduces a taxonomy of activities within FuS-KG and categorizes activities based on the effort required to perform them, detailing their classification and metabolic impact. The module divides the [Activity](https://ida-fbk.github.io/FuS-KG/documentation/html-docs/modules/Core/index-en.html#Activity) concept into two main categories: - [LifeActivity](#LifeActivity): Daily activities essential for living. - [PhysicalActivity](#PhysicalActivity): Activities requiring physical exertion. Each activity is further specialized through the [specializes](#specializes) property, linking it to a [SpecializedActivity](#SpecializedActivity), which defines the metabolic effort required based on the performer. The module also introduces connections to individuals performing these activities: - The [practicedBy](#practicedBy) property associates a [SpecializedActivity](#SpecializedActivity) with a [PractionerType](#PractionerType), identifying the type of person performing the activity (e.g., adult, older adult, wheelchair user). - The metabolic effort is quantified using two properties: - [hasMETValue](#hasMETValue): Specifies the Metabolic Equivalent of Task (MET) coefficient for the activity. - [hasUnitKCalValue](#hasUnitKCalValue): Represents the calories required per minute per kilogram of body weight to perform the activity.
